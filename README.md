sbemoored
=========

SBE moored instrument (SBE56 & SBE37) data processing.

Features
--------
- Convert .cnv/.csv-files to netcdf format.
- Apply clock-drift correction.
- Calculate derived variables using [GSW-Python](https://teos-10.github.io/GSW-Python/)
